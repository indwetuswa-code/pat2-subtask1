morse code is a method of communication that uses sequences of dots{.} and dashes[-] to represent letters,numbers and punctuation. it was one of the earliest forms of digital communication, allowing messages to be sent quickly over long distance using sound, light or electrical signal.
morse code was developed in the 1830s and 1840s by Samuel Morse and Alfred Vail. it was originally designed for use with telegraph system, enabling operators to transmit messages across wires. morse code was vital for communication in military, maritime, and aviation in time of before widespread use of radios and satellites.
Each character{A-Z,0-9} is represented by a unique combination of dots and dashes.  
short signals = dots(.)
long signal = dash (-)
space separates letters and words.
A = .-
B = -...
SOS = ...---...
{ morse code can be comunicated using sound, light and electrical pulse such telegraph line flashes and beeps and tone}
